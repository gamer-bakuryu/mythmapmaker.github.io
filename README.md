🗺️ Forge of Realms | RPG Battlemap Engine
O Forge of Realms é uma ferramenta web de cartografia de última geração para mestres e jogadores de RPG. Unindo a estética Glassmorphism com o poder de renderização de alta fidelidade, o software permite a criação de mapas imersivos em 4K com iluminação dinâmica em tempo real.

✨ Destaques do DesignEstética Hi-Tech Dark:
  Interface profissional com acabamento em vidro jateado para reduzir a fadiga visual.
  Canvas 4K Dinâmico: Renderização de texturas e sombras projetadas inspirada na Unreal Engine 5.
  Workflow Adobe-Style: Sidebar minimalista com controle de camadas (layers) e ferramentas de arquitetura de precisão.
  Seletor de Biomas: Transição rápida entre universos Medieval, Sci-Fi, Cyberpunk e Apocalíptico.

🛠️ Tecnologias Sugeridas
  Frontend: React.js / Next.js
  Canvas/Gráficos: Three.js / WebGL / PixiJS
  Estilização: Tailwind CSS (com filtros de Backdrop Blur)
  Icons: Lucide-React / FontAwesome
