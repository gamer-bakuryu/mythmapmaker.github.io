🗺️ Forge of Realms | RPG Battlemap Engine
O Forge of Realms é uma ferramenta web de cartografia de última geração para mestres e jogadores de RPG. Unindo a estética Glassmorphism com o poder de renderização de alta fidelidade, o software permite a criação de mapas imersivos em 4K com iluminação dinâmica em tempo real.
